V1PER Documentation

V1PER is a toolkit for developers on ROBLOX, avalible for anyone.


Datastores

You can create a new set of leaderstats by creating a new Server Script in ServerScriptService! Then, require V1PER !


Then, we can use our 'BuildNewLeaderstats' function!


We can use this example to create some saving and loading leaderstats which, if you never played, will give you 200 coins!

        
          local V1PER = require(game.ReplicatedStorage.V1PER)

          game.Players.PlayerAdded:Connect(function(Player)
          	local data = V1PER.DataStoreResource.Builder.BuildNewLeaderstats(
          		{
          			Name = 'Coins',
          			TypeValue = 'Int',
          			Deafult = 200,
          		}
          		, Player)

          	V1PER.DataStoreResource.DataStore.LoadData(Player, data)
          end)

          game.Players.PlayerRemoving:Connect(function(Player)
          	V1PER.DataStoreResource.DataStore.SaveData(Player)
          end)
